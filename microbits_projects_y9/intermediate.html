Welcome to Intermediate Level!

You've mastered the basics - now it's time to challenge yourself! These projects introduce more complex programming concepts like radio communication, functions, and data handling. You'll also learn to connect external components to expand your Micro:bit's capabilities.

Skills You'll Learn:

  • Radio communication between Micro:bits
  • Working with external components (speakers, sensors)
  • Creating and using functions
  • Working with lists/arrays
  • More complex conditional logic
  • Data collection and analysis
  • Scientific investigations

Prerequisites

Before starting these projects, make sure you've completed the Beginner level projects or are confident with: displaying on LEDs, using buttons, variables, basic loops, and using sensors.
