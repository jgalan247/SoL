Welcome to Beginner Level!

These projects are perfect for students who have never programmed before. You'll learn the basics of coding while creating fun, interactive projects with your Micro:bit. Each project builds on the previous one, so work through them in order for the best learning experience.

Skills You'll Learn:

  • Using the MakeCode editor
  • Displaying images and text on the LED screen
  • Using buttons for input
  • Basic loops and sequences
  • Introduction to variables
  • Using the accelerometer (shake, tilt)
  • Reading sensor data (temperature, light)
